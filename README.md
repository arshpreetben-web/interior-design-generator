# 🏡 Interior Design Generator (GenAI)

## 📌 Overview

This project uses Generative AI to transform interior design images using Stable Diffusion (SDXL Turbo).

It allows users to upload an image and generate improved interior designs using AI.

---

## 🚀 Features

* Image-to-image generation
* Fast inference with SDXL Turbo
* Simple UI using Gradio
* GPU support

---

## 🛠️ Tech Stack

* Python
* PyTorch
* Diffusers
* Gradio

---

## 📂 Project Files

interior_design_generator.ipynb

---

## ▶️ How to Run

1. Open notebook in Google Colab
2. Enable GPU
3. Run all cells
